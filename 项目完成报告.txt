四、如何构建APK
========================================

【方法一：使用Android Studio（推荐）】
1. 下载安装Android Studio
   https://developer.android.com/studio

2. 打开项目
   File -> Open -> 选择项目文件夹

3. 等待Gradle同步完成

4. 构建APK
   Build -> Build Bundle(s) / APK(s) -> Build APK(s)

5. APK位置
   app/build/outputs/apk/debug/app-debug.apk

【方法二：使用命令行】
前提：已安装Android SDK和JDK

Windows:
cd C:\Users\wd_al\Desktop\ESP32_Debugger_App
gradlew.bat assembleDebug

Linux/Mac:
cd /path/to/ESP32_Debugger_App
./gradlew assembleDebug

【方法三：使用自动构建脚本】
双击运行: build_apk.bat

【方法四：在线构建】
- 上传到GitHub
- 使用GitHub Actions
- 或使用Replit等在线IDE

详细说明请查看: 如何构建APK.md

========================================
五、安装和使用
========================================

【安装步骤】
1. 将APK传输到OnePlus 8T
2. 在设置中启用"未知来源"安装
3. 点击APK文件安装
4. 首次运行时授予权限

【使用WiFi调试】
1. 确保手机和ESP32在同一WiFi网络
2. 在ESP32上运行WiFi服务器代码
3. 在APP中输入ESP32的IP和端口
4. 点击"连接"
5. 发送和接收数据

【使用蓝牙调试】
1. 打开手机蓝牙
2. 在ESP32上启动蓝牙服务
3. 在APP中点击"扫描"
4. 选择ESP32设备
5. 点击"连接"
6. 发送和接收数据

Q: 构建需要多长时间？
A: 首次安装Android Studio: 30-60分钟
   首次构建项目: 10-20分钟
   后续构建: 1-3分钟

Q: 可以在其他手机上使用吗？

Q: 如何修改代码？
A: 使用Android Studio打开项目，

Q: 支持哪些ESP32型号？

   只要支持WiFi或蓝牙功能。
